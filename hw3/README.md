## Preparation
In your original dpm-f23 directory, `git pull` to get new `hw3` directory.


## Requrirements

The directory should be look like this.

```
.
├── enums.py
├── main.py
├── map.png
├── nav.py
├── pointcloud
│   ├── color01.npy
│   ├── color0255.npy
│   └── point.npy
├── README.md
├── replica_v1 -> /data/replica_v1
├── results
│   ├── *.mp4
│   └── *.png
└── semantic_3d_pointcloud
    ├── color01.npy
    ├── color0255.npy
    └── point.npy
```
